# Webpage_Text_Extraction

Extract main textual information from HTML.

## Requirement

* Python 3
* requests
* beautifulsoup4

## Installation

1. Download this repository.
2. Change current directory into this folder.   
3. Installation.

    ```shell
    pip install .
    ```
